# 页面规格

## 1. 信息架构

底部导航建议 3 个 Tab：

- 首页。
- 换算。
- 设置。

统计可以先放在首页下方或设置页入口，首版不必单独做 Tab。

## 2. 首页

页面目标：

- 一眼看到今天赚了多少钱。
- 一眼知道多久下班。
- 快速进入消费换算。

核心区域：

- 状态标签：工作中 / 午休中 / 未上班 / 已下班 / 休息日。
- 主数字：今日已赚金额。
- 副数字：距离下班。
- 进度条：今日工作进度。

辅助区域：

- 今日剩余可赚。
- 今日已工作时长。
- 当前分钟工资。

快捷操作：

- 消费换算。
- 编辑工资。
- 开启加班，Pro。

## 3. 消费换算页

页面目标：

- 快速输入金额。
- 立即得到工作时间换算。

组件：

- 金额输入框。
- 常用消费按钮。
- 结果卡片。
- 最近记录列表。

结果文案示例：

- 这笔消费需要你工作 42 分钟。
- 约等于今天已赚收入的 23%。
- 你今天已经赚够 3.2 杯咖啡。

## 4. 设置页

分组：

- 工资设置。
- 工作时间。
- 午休设置。
- 工作日。
- 主题和小组件。
- Pro。
- 关于。

工资设置字段：

- 工资模式。
- 工资金额。
- 货币单位。
- 发薪日。

工作时间字段：

- 上班时间。
- 下班时间。
- 午休开始。
- 午休结束。

## 5. Pro 页面

页面目标：

- 让用户明确知道付费后能得到什么。

展示顺序：

1. 当前触发功能的解锁说明。
2. Pro 权益列表。
3. 价格选项。
4. 购买按钮。
5. 恢复购买、服务条款、隐私政策。

权益文案：

- 解锁全部小组件样式。
- 自定义主题和图标。
- 管理多份工作和兼职。
- 查看月度收入统计。
- 记录消费回本。
- 计算加班收入。

## 6. 小组件规格

小号：

- 今日已赚金额。
- 状态短文案。

中号：

- 今日已赚金额。
- 下班倒计时。
- 进度条。

样式建议：

- 默认简洁白底。
- 深色模式。
- 高对比数字。
- 不放过多说明文字。
