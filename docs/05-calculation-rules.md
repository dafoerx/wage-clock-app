# 计算规则

## 1. 基础定义

有效工作时间：

```text
每日有效工作分钟 = 下班时间 - 上班时间 - 午休分钟
```

分钟工资：

```text
分钟工资 = 时薪 / 60
```

今日已赚：

```text
今日已赚 = 今日已工作有效分钟 * 分钟工资
```

## 2. 月薪模式

输入：

- 月薪金额。
- 每周工作日。
- 每日有效工作分钟。

计算：

```text
当月工作日数量 = 当前月份中被选择为工作日的日期数量
当月有效工作小时 = 当月工作日数量 * 每日有效工作分钟 / 60
时薪 = 月薪 / 当月有效工作小时
分钟工资 = 时薪 / 60
```

今日已赚：

```text
今日已赚 = 今日有效工作分钟 * 分钟工资
```

## 3. 日薪模式

输入：

- 日薪金额。
- 每日有效工作分钟。

计算：

```text
分钟工资 = 日薪 / 每日有效工作分钟
今日已赚 = 今日有效工作分钟 * 分钟工资
```

## 4. 时薪模式

输入：

- 时薪金额。

计算：

```text
分钟工资 = 时薪 / 60
今日已赚 = 今日有效工作分钟 * 分钟工资
```

## 5. 今日有效工作分钟

不同状态：

- 未上班：0。
- 工作中：从上班时间到当前时间，扣除已经发生的午休时间。
- 午休中：从上班时间到午休开始。
- 已下班：每日有效工作分钟。
- 休息日：0。

午休处理：

```text
如果当前时间早于午休开始：不扣午休
如果当前时间在午休中：扣除当前时间 - 午休开始
如果当前时间晚于午休结束：扣除完整午休
```

首版也可以简化为：

- 午休中收入暂停。
- 午休结束后扣除完整午休。

## 6. 下班倒计时

工作日：

```text
如果当前时间早于上班时间：显示距离上班
如果当前时间在工作时间内：显示距离下班
如果当前时间晚于下班时间：显示已下班
```

休息日：

```text
显示今天休息
```

## 7. 消费换算

输入：

- 消费金额。

计算：

```text
需要工作分钟 = 消费金额 / 分钟工资
```

展示：

- 小于 60 分钟：显示 N 分钟。
- 大于等于 60 分钟：显示 X 小时 Y 分钟。
- 大于等于 1 个工作日：显示 X 天 Y 小时。

## 8. 加班收入

首版可以作为 Pro 功能。

输入：

- 加班开始时间。
- 加班结束时间。
- 加班倍率：1x、1.5x、2x、3x。

计算：

```text
加班收入 = 加班分钟 * 分钟工资 * 加班倍率
```

## 9. 金额展示

建议：

- 首页主金额保留 2 位小数。
- 小组件可保留 1 位或不显示小数。
- 换算结果用近似文案，避免制造精度焦虑。

示例：

- ¥128.56。
- 约 ¥129。
- 约 42 分钟。
