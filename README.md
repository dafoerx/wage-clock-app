# Wage Clock App

这是一个面向 iOS App Store 的轻量工具 App 方案，核心方向是：

> 工资计时 + 下班倒计时 + 消费换算 + 桌面小组件。

产品目标是用尽量简单的 MVP 验证用户是否愿意为上班收入实时展示、小组件、主题和统计功能付费。

## 当前建议

首版建议先采用：

- 免费下载。
- 本地数据，无需登录。
- 免费版可完整体验 1 个工作配置。
- Pro 使用一次性买断或低价年付先验证。

早期不建议一上来做复杂订阅、账户系统、社交、银行同步、自动考勤或完整记账。
